🔑 API Key Setup
One-time setup per device. Your key stays saved in this browser.
How to get your API key
1
Go to the Anthropic Console

Open a new tab and go to console.anthropic.com — sign in with the same email you used to set up your Anthropic account (likely your personal email or jerrod@fiveanddive.ca).

2
Go to API Keys

In the left sidebar, click API Keys. If you already created one before, copy it from wherever you stored it (notes app, email, etc.) and skip to step 4.

3
Create a new key

Click "Create Key". Give it a name like FiveDive Studio so you remember what it's for. Click Create.

⚠️ Copy it immediately — Anthropic only shows the key once. You cannot retrieve it after closing that page.
4
Paste it above & save

Paste the key into the field at the top of this window. It starts with sk-ant-. Hit Save Key — done. You won't need to do this again on this device and browser.

5
Lost your key or it stopped working?

Old keys can't be recovered — but creating a new one takes 30 seconds. Go to console.anthropic.com/settings/keys, delete the old one, create a new one, and paste it here again.

🔒 Your key stays on your device. It's saved in this browser's local storage only — never stored anywhere else. It goes directly to Anthropic's API when generating content and nowhere else.
