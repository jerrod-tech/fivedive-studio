Layout Rules — Ryan's System
Extracted from 30+ actual posters. These are non-negotiable.
  • Background IS the event — fire for Fireball, mic for Open Mic, bingo balls for Bingo, roses for spring, dragon for D&D. Always a real thematic photo, never a plain gradient.
  • FIVE&DIVE logo always top-center (vertical posters) or top-left (horizontal). "FOOD ✦ BOOZE ✦ ROCK'N'ROLL" immediately below in small caps with arrow separators.
  • Event title takes 40–60% of the poster in NewRocker. White fill, thick black outline. It dominates everything.
  • Frequency badge ("1st Thursday Of The Month!" / "WEDNESDAYS") sits just above the main title in red script or a grey pill.
  • "Hosted By" in small Dancing Script → host name in NewRocker below it. Always paired as a unit.
  • Time ("Starts @ 7:30pm!") in red Dancing Script, below the hosted-by unit.
  • Dumpster Dives special ("$2 off / Dumpster Dives / After 7pm!") appears on ALL Thursday promos. Gold grunge for $2, script for the rest.
  • Date chips on Live Music Weekend calendar: cream/tan rectangles with big date number + small month text. NOT red badges.
  • Lightning bolts ⚡ flank the title on Battle of the Bands and Fireball posters. Yellow on dark bg.
  • Sponsor logos always bottom-center in a row, just above the footer bar.
  • Bottom bar: phone left · Facebook + Instagram icons center · website right. Always. Every poster.
  • Yellow pill badge for special CTAs: "Starts August 28th!" / "Want Your Band To Join?" — black text on bright yellow.
  • Font matches the event personality — GTA font for Games & Trivia, Gothic for rock events, script-heavy for casual nights.
  • Grain/texture overlay on everything, even bright sunburst posters. Never perfectly clean.

🎨 Instagram Image Styles — Ryan's Spectrum
Style 1 — Artist Photo Full Bleed (Troy Turner, El Niven): Real band/artist photo, gothic name overlay, date. Blue/teal or dark tones. Most common for live music.

📺 Budvue Upload Steps
  1. Log in at app.budvue.com → Media Library
  2. Click Upload → select the 1920×1080 TV PNG from Ryan
  3. Name it: F_D-[Event]-TV_1920x1080-R1
  4. Go to Playlists → Live Music → drag new slide to position
  5. Set display duration 15 sec · click Save & Publish
  6. Walk to a TV to confirm — or screenshot the preview panel
🌐 Squarespace Calendar Steps
  1. Go to fiveanddive.ca → click Edit (top bar)
  2. Navigate to the Calendar page → click the event date
  3. Click Edit Event → Content tab
  4. Paste the event title, date, time (9:30pm), and description
  5. Click Options tab → upload the 1080×1080 PNG as featured image
  6. Click Save → Done → Publish
